Another Controller for ManiaPlanet in C

Welcome to the project page of this Maniaplanet Controller named AYO.

AyO ( not the official name yet ) is another controller for ManiaPlanet in C# based on SMAdminTools.

AyO features :

  • Fully C# supported.
  • Dynamic manialinks.
  • Create Manialinks like you do in ManiaPlanet.
  • Relay server for communication between server and client ( for sending var to the client for example ).
  • AyO include a powerful plugin named "Commands".
  • Creating plugins is easy.
  • Full XmlRpc methods support.
  • For Windows/Mac and Linux servers.
  • Works in Local
  • Fast build.
  • And some features from SMAdminTools :)
